Для начала надо установить два модуля: requests(pip install requests), bs4(pip install bs4).
Дальше, закидываем скрипт с модулем в одну папку со скриптом, в котором вы хотите его юзать.
В скрипте вначале прописываем from parsing import Url (если вы меняли название файла с модулем, вместо parsing надо написать название файла)
Затем создаем экземпляр класса Url, в него в качестве аттрибута передаем юрл сайта для парсинга, можно передать ваш юзерагент, если он не будет передан, будет использоваться дефолтный юзер агент (urlObj = Url(юрл, user_agent=ваш_юзер_агент))
В конце создадим переменную для хранения результата парсинга, ее значением будет возращение функции parse, в которую в качестве аргументов надо передать класс элемента и его тег (result = urlObg.parse(класс, тег))
Переменная result - это строка, с которой можно будет работать.
